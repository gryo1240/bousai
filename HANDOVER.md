### アプリの特徴(記事の訴求ポイント)

1. **家族構成を入れるとリストが変わる**: 大人・子ども・乳幼児・高齢者・ペットの人数を+/-で設定すると、必要な品目(最大56品目)と目安数量を自動計算。「飲料水は3人なら9本」「簡易トイレは45回分」のように具体的な数字が出る
2. **乳幼児・高齢者・女性・ペットの「見落としがちな備え」に対応**: 液体ミルク・おむつ(サイズアウト注意)・入れ歯・生理用品・ペットフード5日分など、一般的なリストで抜けやすい物を家族構成に応じて自動追加
3. **チェックすると準備率(%)が見える**: ゲージが伸びるので「あとどれくらい」がわかり、進めるモチベーションになる
4. **足りない物だけ買い物リストに**: 未準備の物だけをテキストコピー。ほとんど100均・ドラッグストアで揃う
5. **「作って終わり」にしない知識5本**: 持ち出しリュックと備蓄の違い(全部背負わない)・置き場所は玄関か寝室・年2回の点検日・家族の集合場所と災害用伝言ダイヤル171・入れすぎと見落としの定番
6. データは端末内のみ・登録不要

### 想定読者(記事の切り口)

「防災リュック、用意しなきゃと思いながら何年も経っている」人。記事リストを眺めるだけで終わりがちな防災準備を、**チェックできるアプリにすることで「今日から始められる」**のが差別化。乳幼児・ペットがいる家庭は特に既製の防災セットだけでは足りないので、家族構成カスタムが刺さる。9月1日(防災の日)・3月11日前後は毎年検索が急増する。

## 記事の下書き(そのまま使用・編集可)

タイトル案:

- 「『いつかやろう』の防災準備を今日終わらせる。チェック式の無料アプリ『防災リュックチェッカー』」

---

**防災リュック、「用意しなきゃ」と思ったまま何年も経っていませんか。**

私もそうでした。ネットでリストを調べても、眺めて閉じて終わり。「防災リュックチェッカー」は、その"いつかやろう"を今日終わらせるための無料アプリです。

**家族構成を入れると、わが家専用のリストになる**

大人2人・子ども1人・赤ちゃん1人・ペット1匹——と設定すると、必要な物と数量が自動で計算されます。「飲料水は9本」「簡易トイレは45回分」「おむつは約15枚」。一般的なリストでは抜けがちな、液体ミルク・入れ歯・生理用品・ペットフードなども、家族構成に応じて自動で加わります。

**チェックするたび、準備率が上がっていく**

持っている物にチェックを入れると「わが家の備え 64%」のようにゲージが伸びます。ゼロから完璧を目指さなくて大丈夫。まず「足りない物」タブを開くと、未準備の物だけの買い物リストができて、そのままコピーできます。ほとんどは100均とドラッグストアで揃います。

**いちばん大事なのは「全部リュックに入れない」こと**

水も食料も全部背負うと、重くて逃げられません。アプリの知識タブには「持ち出しリュックと備蓄は別物」「リュックは玄関か寝室に」「年2回の点検日を決める」「家族の集合場所と伝言ダイヤル171」など、防災士の間で定番とされる基本を収録。リストを揃えて終わりではなく、"使える備え"になるように作っています。

**インストール方法(無料・登録不要)**

- Android: [アプリを開く](https://gryo1240.github.io/bousai/) → Chromeメニュー →「アプリをインストール」
- iPhone: Safariで開く → 共有ボタン →「ホーム画面に追加」

チェック状況は端末の中にだけ保存されます。

## アプリ側を修正したい場合

- 本体は `index.html` 1ファイル。`main` に push で GitHub Pages に自動デプロイ
- 品目は `ITEMS` 配列(id/cat/name/grp/q/s)。`grp` が null なら全員、"baby"|"kid"|"senior"|"woman"|"pet" なら該当家族がいる時のみ表示。`q(fam)` が数量文字列を返す(基準人数は `ppl()`=大人+子ども+高齢者)
- **リライト不要の設計方針**: 価格・商品名・制度など変わりうる情報を持たない。品目追加時もこの方針を守ること
- 知識記事は `ARTICLES` 配列
- localStorageキー: `bousai-v1`(fam+have)。同期機能はなし(全データ端末内)
- sw.js のキャッシュ名は `bousai-` プレフィックス(他アプリのキャッシュを消さないよう activate で自プレフィックスのみ削除)
