# 防災リュックチェッカー

家族構成に合わせた防災リュック(一次持ち出し品)の中身リストを自動作成して、準備状況をチェックできる無料のWebアプリ(PWA)。

**アプリ本体: https://gryo1240.github.io/bousai/**

- **家族構成でリストが変わる**: 大人・子ども・乳幼児・高齢者・ペットの人数と「女性がいる」を設定すると、必要な持ち出し品(約40〜56品目)と目安数量を自動計算(例: 飲料水は人数×3本、簡易トイレは人数×15回分)
- **持っている物をチェック**: カテゴリ別(水・食料/情報・電源/衛生・トイレ/救急/貴重品/衣類/道具+条件別グループ)のチェックリストで準備率(%)を表示
- **足りない物だけ買い物リスト**: 未準備の物だけを抽出してテキストコピー。100均・ドラッグストアで揃う物が中心
- **「作って終わり」にしない知識**: 持ち出しと備蓄の違い・リュックの置き場所・年2回の点検・家族の集合場所と171など、普遍的な防災知識5本を収録
- **リライト不要の設計**: 価格・商品名・制度情報を一切持たず、数量計算と普遍的な知識のみで構成
- チェック状況・家族構成は端末内(localStorage)のみ保存。外部送信なし・登録不要

## 構成

| ファイル | 役割 |
|---|---|
| `index.html` | アプリ本体(HTML/CSS/JSすべて1ファイル。品目データ`ITEMS`を内蔵) |
| `manifest.webmanifest` / `sw.js` | PWA化(インストール・オフライン対応) |
| `icon-*.png` / `apple-touch-icon.png` | アプリアイコン |
| `docs/screenshot-*.png` | 紹介用スクリーンショット |

`main` ブランチに push すると GitHub Pages に自動デプロイされます。

ブログ等での紹介記事を書く場合は [HANDOVER.md](HANDOVER.md) を参照してください。
